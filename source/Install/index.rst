Установка Arenadata Streaming
===============================

В документе приведен принцип установки Arenadata Streaming c помощью Ambari 2.7.0 и порядок подготовки к установке Arenadata Streaming MPack.

Документ может быть полезен администраторам, программистам, разработчикам и сотрудникам подразделений информационных технологий, осуществляющих внедрение платформы.
