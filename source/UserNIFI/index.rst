Руководство пользователя по работе с сервисом Nifi
=====================================================

В руководстве приведены сведения для пользователей системы по работе с платформой ADS в части серсиса Nifi – рекомендации по конфигурации, аутентификация пользователей с настройками их политик и доступа, Kerberos.

Руководство может быть полезно администраторам, программистам, разработчикам и сотрудникам подразделений информационных технологий, осуществляющих сопровождение платформы.
